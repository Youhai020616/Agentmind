# 商业化方案与定价

## 市场分析

### 目标市场

```
TAM (Total Addressable Market)
├── 全球 AI Agent 用户 ≈ 2000 万开发者 (2026)
│
├── SAM (Serviceable)
│   ├── 使用 Claude Code / Cursor / Copilot 的开发者 ≈ 500 万
│   └── 有团队协作需求 ≈ 200 万
│
└── SOM (Obtainable, Year 1)
    ├── 开发者工具早期采用者 ≈ 1-5 万
    └── 目标 ARR: $50K - $500K
```

### 竞品分析

| 竞品 | 定位 | 收费 | 差异 |
|------|------|------|------|
| Cursor Memory | 编辑器内置 | 免费(捆绑) | 仅限 Cursor 平台 |
| Mem0 | 通用 AI 记忆 | 开源+云 | 偏向对话记忆，非开发专用 |
| LangMem | LangChain 记忆 | 开源 | 偏向 LLM 应用开发 |
| **AgentMind** | **Agent 学习中间件** | **免费+SaaS** | **开发者专用，四层观察，进化系统** |

### AgentMind 的护城河

1. **四层观察** — 竞品只能观察一层
2. **效果反馈闭环** — 竞品只能记录，不验证
3. **进化抽象** — 从碎片到体系的自动升级
4. **平台无关** — 不绑定特定 Agent
5. **开发者社区** — 本能市场 + 开源核心

## 产品分层

### Free（开源核心）

```
适合：个人开发者
价格：$0

功能：
✅ 本地观察 + 分析
✅ 本能创建与管理
✅ CLI 工具
✅ 基础置信度（频率分）
✅ 简单聚类进化
✅ 单项目
✅ Claude Code + Cursor 适配器

限制：
❌ 无云同步
❌ 无团队功能
❌ 无效果验证（需要 LLM API）
❌ 无高级进化（抽象提升）
❌ 无 Dashboard
```

### Pro（$12/月 个人订阅）

```
适合：重度开发者
价格：$12/月 或 $99/年

功能：
✅ Free 所有功能
✅ 云同步 + 多设备
✅ Web Dashboard
✅ 复合置信度（三维评分）
✅ 效果验证（内置 Haiku 分析）
✅ 高级进化（抽象提升 + 迁移）
✅ 最多 5 个项目
✅ 本能导入/导出
✅ 每月 1000 次 LLM 分析额度

限制：
❌ 无团队功能
❌ 无本能市场
```

### Team（$8/用户/月 团队订阅）

```
适合：开发团队
价格：$8/用户/月（最少 3 人）

功能：
✅ Pro 所有功能
✅ 团队本能共享
✅ 团队 Dashboard + 分析
✅ 项目级本能管理
✅ 新成员自动继承团队本能
✅ 审核流程（Leader 审核团队本能）
✅ 无限项目
✅ 每用户每月 3000 次分析额度
✅ 本能市场访问（发布 + 下载）

限制：
❌ 无 SSO
❌ 无审计日志
❌ 无 SLA
```

### Enterprise（联系销售）

```
适合：大型组织
价格：联系销售

功能：
✅ Team 所有功能
✅ SSO (SAML/OIDC)
✅ 审计日志
✅ 自托管选项
✅ 私有本能市场
✅ 专属客户经理
✅ SLA 99.9%
✅ 合规认证 (SOC2, GDPR)
✅ 无限 LLM 分析额度
✅ 定制适配器开发
```

│              收入来源                         │

│     ├── 超额 LLM 分析：$0.001/次             │
│     └── 超额存储：$0.01/GB/月                │

### 本能市场

```
开发者 A 的进化产物                     开发者 B 购买/下载
"React TypeScript 最佳实践"    →     导入为 inherited 本能
包含：15 本能 + 3 模式 + 1 策略         初始置信度 0.6

定价：
  ├── 免费：个人分享的小本能包
  ├── $5-$15：领域专家整理的本能包
  ├── $20-$50：完整专家系统包
  └── 按需定价：企业级行业方案

平台抽成：30%（苹果模式）
创作者收入：70%
```

## 关键指标

| 指标 | 含义 | 目标 |
|------|------|------|
| **CAC** | 获客成本 | < $50 (Pro), < $500 (Team) |
| **LTV** | 客户生命周期价值 | > $200 (Pro), > $2000 (Team) |
| **LTV/CAC** | | > 4x |
| **MRR** | 月度经常性收入 | Year 1: $5K, Year 2: $50K |
| **Churn** | 月流失率 | < 5% (Pro), < 3% (Team) |
| **NRR** | 净收入留存 | > 110% |

## 定价心理学

| 策略 | 实现 |
|------|------|
| **锚定效应** | 先展示 Enterprise（无价格），再展示 Pro（$12 显得便宜） |
| **免费试用** | Pro 14 天免费试用，Team 30 天免费 |
| **年付优惠** | 年付 = 10 个月价格（省 17%） |
| **团队优惠** | 人数越多单价越低（>20 人 $6/人，>50 人 $5/人） |
| **学生免费** | .edu 邮箱免费 Pro（培养未来付费用户） |
| **开源贡献者** | 有贡献的开源开发者免费 Pro |

## 风险与对策

| 风险 | 概率 | 对策 |
|------|------|------|
| Agent 平台内置类似功能 | 高 | 保持平台无关性 + 深度差异化 |
| 开源竞品 | 中 | 社区领先 + 云服务差异化 |
| 隐私担忧 | 中 | 本地优先 + 端到端加密 + 合规认证 |
| LLM API 成本上升 | 低 | 多 LLM 支持 + 本地小模型备选 |
| 用户留存低 | 中 | 效果可视化 + 量化 ROI |
